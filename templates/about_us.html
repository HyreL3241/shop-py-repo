Project Overview

This platform was developed as part of our Software Design course midterm project.

Our objective was to build a fully functional e-commerce site showcasing best practices in UI/UX, database design, modular code structure, and collaborative teamwork within an agile workflow.

Technology Stack
  • Backend: Django (Python)
  • Frontend: HTML5, CSS3, Bootstrap, JavaScript
  • Database: SQLite (Django ORM)
